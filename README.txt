Basic list of features:

- Gameboy, Gameboy Color, Super Gameboy emulation
- Supports Gameboy Color Bios ("boot rom") for custom palettes
- Emulates Gameboy Color palettes for Gameboy games if bios not present
- Auto SRAM saving
- Save States
- Cheat Codes
- Remappable controls
- Custom borders
- Custom palettes for GB games
- Color correction approximating original GBC screen
- Scale to fill the screen
- Gameboy Printer emulation
- Gameboy Camera emulator (DSi only)
- GBS music playback
